# 官方 3.5.6 复刻对比简报（cloud-only）

> 状态：**范围归档**
> 更新：2026-07-12
> 说明：旧版包含 BLE/QGJ 和真机验证对比，相关内容已取消并从正文移除。

## 当前结论

本项目定位为官方云端控车工具，不以完整复刻官方平台型 App 为目标。

已覆盖的核心路径：

- 官方账号登录和会话保持
- 官方车辆列表、详情、状态与多车切换
- 云端设防、解防、上电、断电、寻车和开坐垫
- 消息、通知偏好、定位、轨迹、围栏和电池信息
- 骑行统计、日志、诊断和本地偏好

仍属于 cloud-only 产品待办的项目：

- 官方车辆昵称回写
- 家庭共享官方授权闭环
- 多语言资源与业务文案切换
- 新增云 API 的自动化边界测试

## 明确不做

以下差距不计入复刻进度，也不作为发布阻断项：

- 本地硬件直连、扫描、GATT、感应解锁和离线控车
- 扫码、IMEI、门店绑定、解绑和车辆转让
- NFC 真加钥匙、投屏、智能仪表和摄像头能力
- OTA、TPMS、高级 ECU 或 BMS 写入
- 充电交易、支付、保险、商城、积分、社区和售后闭环
- 依赖实体手机、实体车辆或真实车型覆盖的回归测试
