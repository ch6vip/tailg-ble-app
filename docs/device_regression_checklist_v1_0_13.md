## 0. 环境与构建

### 0.1 设备
- [ ] Android 真机（建议 1 台日常机 + 可选 1 台弱网机）
- [ ] 系统时间正确；可连外网访问官方云 API
- [ ] 有可用的台铃官方账号（短信可收验证码）
- [ ] 账号下至少 1 辆已绑定智能车（可云控）
- [ ] 可选：同一账号第二辆车（多车切换）
- [ ] 可选：空账号或解绑后无车账号（测 unbound）

### 0.2 构建安装
- [ ] 使用含 P0 + P0.5 的最新代码打 debug/release APK
- [ ] 卸载旧包或确认覆盖安装成功
- [ ] 冷启动不崩溃；启动失败页不出现
- [ ] 底部 Tab 为：服务 / 爱车 / 我的（无圈子/商城）

| ID | 步骤 | 期望 | ☐ |
| --- | --- | --- | --- |
