## UI 优化 — 6 大类 14 项修复

基于 UI 分析报告，完成了以下修改：

### P0 高优先级
- **`status_badge.dart`**: _PulsingDot 只在 armed/ble/online 活跃状态脉冲，idle/offline 静态
- **`control_card.dart`**: _SideButton→StatefulWidget，按下 AnimatedScale(0.96) + disabled AnimatedOpacity(0.45)
- **`control_card.dart`**: _SubControl→StatefulWidget，同理按压反馈
- **`control_card.dart`**: ControlCard 移除 Opacity(0.55) 包装；_PowerKnob busy 状态改为 pulsing glow ring + CircularProgressIndicator + "处理中…"文字

### P1 中优先级
- **`control_page_hero.dart`**: _TopIconButton 36→44px，加 HitTestBehavior.opaque，满足 WCAG 2.5.5
- **`control_page_mode_widgets.dart`**: Colors.blue → AppColors.accentSky
- **`app_colors.dart`**: 添加绿色 token 映射注释(info/accentTeal 是 success 别名)；新增 AppColorsData/AppColorsLight/AppColorsDark 暗色模式 token 集；添加 AppColors.of(context) 工厂方法
- **`app_motion.dart`** (新建): AppMotion 动效规范 — pressScale/micro/standard/pressCurve 等常量和建议值

### P2 低优先级
- **`control_page_service_cards.dart`**: toggle 假数据项加锁图标 + Tooltip("功能开发中，暂不可用")
- **`control_page_unbound_home.dart`**: _UnboundBanner→StatefulWidget，PageView 3 页自动轮播(4s)；_UnboundVehicleHome 支持 connectionLost 参数显示"连接已中断"重试界面
- **`control_page.dart`**: 检测 connectionLostHint 传递给 _UnboundVehicleHome
